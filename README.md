## 操作手册
shift+左键可以选择一个圆形

鼠标左键按住可以吸引一个被选择的圆形

f：切换固定状态，可以将活动的圆形设置为固定，将固定的圆设置为可活动

h：生成一个由圆组成的弹性体矩形，参数可调

z：删除上一个圆
ctrl+z：开始清除流体粒子
shift+z：删除上一个用边界粒子画出来的线
shift+w：弹性矩形宽度增加
ctrl+w：弹性矩形宽度减小

shift+s：弹性矩形高度增加
ctrl+s：弹性矩形高度减小
